# vue-babylonjs
A Babylon.js integration to Vue.js. Sample project using javascript, latest babylon.js es6, eslint.

## Project setup

```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
